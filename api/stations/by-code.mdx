## Endpoint

```
GET https://api.terraquakeapi.com/v1/stations/code
```

## Description

Retrieves detailed information for a single seismic monitoring station based on its unique station code. The lookup is case-insensitive, meaning "ACATE", "acate", and "Acate" will all return the same result.

Station codes follow FDSN naming conventions (e.g., "ACATE", "IV.ACER") and uniquely identify each monitoring station in the INGV network.

## Use Cases

- **Station Lookup**: Get complete details for a specific station
- **Validation**: Verify station existence and current status
- **Deep Dive**: Access full metadata for individual stations
- **Integration**: Retrieve station info for event analysis workflows

## Parameters

<ParamField
  query='code'
  type='string'
  required
>
  The unique station code to search for. Case-insensitive. **Examples**:
  "ACATE", "IV.ACER", "MONC" **Format**: Alphanumeric, may include periods for
  network prefix
</ParamField>

## Request Examples

<CodeGroup>

```bash cURL
curl "https://api.terraquakeapi.com/v1/stations/code?code=ACATE"
```

```javascript JavaScript
const stationCode = 'ACATE';
const response = await fetch(
  `https://api.terraquakeapi.com/v1/stations/code?code=${stationCode}`,
);
const data = await response.json();

if (data.success) {
  const station = data.payload[0];
  console.log(`Station: ${station.$.code}`);
  console.log(`Location: ${station.Site.Name}`);
  console.log(`Coordinates: ${station.Latitude}, ${station.Longitude}`);
  console.log(`Elevation: ${station.Elevation}m`);
  console.log(`Status: ${station.$.restrictedStatus}`);
} else {
  console.error(data.message);
}
```

```python Python
import requests

station_code = "ACATE"
url = f"https://api.terraquakeapi.com/v1/stations/code?code={station_code}"

response = requests.get(url)
data = response.json()

if data.get('success'):
    station = data['payload'][0]
    print(f"Station: {station['$']['code']}")
    print(f"Location: {station['Site']['Name']}")
    print(f"Coordinates: {station['Latitude']}, {station['Longitude']}")
    print(f"Elevation: {station['Elevation']}m")
    print(f"Status: {station['$']['restrictedStatus']}")
else:
    print(f"Error: {data.get('message')}")
```

## Case Insensitivity

The code parameter is case-insensitive. All of these requests return the same result:

```bash
# All equivalent
curl "https://api.terraquakeapi.com/v1/stations/code?code=ACATE"
curl "https://api.terraquakeapi.com/v1/stations/code?code=acate"
curl "https://api.terraquakeapi.com/v1/stations/code?code=Acate"
```

## Practical Example

Check if a station is operational before using it for analysis:

```javascript
async function isStationOperational(stationCode) {
  try {
    const response = await fetch(
      `https://api.terraquakeapi.com/v1/stations/code?code=${stationCode}`,
    );
    const data = await response.json();

    if (!data.success) {
      console.log(`Station ${stationCode} not found`);
      return false;
    }

    const station = data.payload[0];
    const isOpen = station.$.restrictedStatus === 'open';

    console.log(
      `Station ${stationCode} is ${isOpen ? 'operational' : 'inactive'}`,
    );
    return isOpen;
  } catch (error) {
    console.error('Error checking station:', error);
    return false;
  }
}

// Usage
await isStationOperational('ACATE');
```
