---
title: Get Last Week's Earthquakes
description: Retrieve all earthquake events from the last 7 days
---

Fetches all earthquake events that occurred in the last 7 days, including today. This endpoint is ideal for weekly summaries and trend analysis.

## Endpoint

**Method:** GET  
**Path:** `/v1/earthquakes/last-week`  
**Authentication:** Not required

## Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination. Must be a positive integer greater than 0.
  
  **Validation:**
  - Must be > 0
  - Returns 400 error: "The 'page' parameter must be a positive integer (e.g., ?page=2)."
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of results per page. Must be a positive integer greater than 0.
  
  **Validation:**
  - Must be > 0
  - Returns 400 error: "The 'limit' parameter must be a positive integer greater than 0. Example: ?limit=50"
</ParamField>

<ParamField query="sort" type="string" default="-time">
  Sort order for results. Prefix with `-` for descending order.
  
  **Allowed values:**
  - `time` or `-time` (default: descending)
  - `magnitude` or `-magnitude`
  - `depth` or `-depth`
</ParamField>

<ParamField query="fields" type="string">
  Comma-separated list of fields to include in response.
  
  **Allowed values:**
  - `time`
  - `magnitude`
  - `depth`
  - `place`
  - `coordinates`
</ParamField>

## Request Example

<CodeGroup>
```bash cURL
curl "https://api.terraquakeapi.com/v1/earthquakes/last-week?limit=200"
```

```javascript JavaScript
const response = await fetch(
  'https://api.terraquakeapi.com/v1/earthquakes/last-week?limit=200'
);
const data = await response.json();

if (data.success) {
  console.log(`Earthquakes in last 7 days: ${data.totalEarthquakes}`);
  console.log(`Date range: ${data.message}`);
  data.payload.forEach(quake => {
    const { mag, place, time } = quake.properties;
    console.log(`${mag} - ${place} at ${time}`);
  });
}
```

```python Python
import requests

response = requests.get(
    'https://api.terraquakeapi.com/v1/earthquakes/last-week',
    params={'limit': 200}
)
data = response.json()

if data.get('success'):
    print(f"Earthquakes in last 7 days: {data.get('totalEarthquakes')}")
    print(f"Date range: {data.get('message')}")
    for quake in data.get('payload', []):
        props = quake['properties']
        print(f"{props['mag']} - {props['place']} at {props['time']}")
```
</CodeGroup>

</ResponseField>

<ResponseField name="code" type="integer">
  HTTP status code (200 for success)
</ResponseField>

<ResponseField name="status" type="string">
  HTTP status message ("OK" for success)
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable message including date range. Format: "Earthquake events from the last 7 days ({startDate} to {endDate})"
</ResponseField>

<ResponseField name="payload" type="array">
  Array of GeoJSON Feature objects containing earthquake data
</ResponseField>

<ResponseField name="meta" type="object">

</ResponseField>

## Implementation Details

- **Date Calculation:** `today.getDate() - 7` to get start date
- **Date Range:** Last 7 days including today (e.g., Oct 30 to Nov 6)
- **Sorting:** Default sort is by time (most recent first: `-time`)
- **Pagination:** Manual pagination applied after fetching all events from INGV
- **Data Source:** INGV API with `orderby=time` and `format=geojson`

- Comparing recent activity to historical data
