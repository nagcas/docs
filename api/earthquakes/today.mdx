---
title: Get Today's Earthquakes
description: Retrieve all earthquake events that occurred today (UTC timezone)
---

Fetches all earthquake events that occurred on the current calendar day in UTC timezone. This endpoint is ideal for daily monitoring and real-time alerts.

## Endpoint

**Method:** GET  
**Path:** `/v1/earthquakes/today`  
**Authentication:** Not required

## Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination. Must be a positive integer greater than 0.
  
  **Validation:**
  - Must be > 0
  - Returns 400 error: "Invalid 'page' parameter. It must be a positive integer (e.g., ?page=2)."
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of results per page. Must be a positive integer greater than 0.
  
  **Validation:**
  - Must be > 0
  - Returns 400 error if invalid
</ParamField>

<ParamField query="sort" type="string" default="-time">
  Sort order for results. Prefix with `-` for descending order.
  
  **Allowed values:**
  - `time` or `-time` (default: descending)
  - `magnitude` or `-magnitude`
  - `depth` or `-depth`
</ParamField>

<ParamField query="fields" type="string">
  Comma-separated list of fields to include in response.
  
  **Allowed values:**
  - `time`
  - `magnitude`
  - `depth`
  - `place`
  - `coordinates`
</ParamField>

## Request Example

<CodeGroup>
```bash cURL
curl "https://api.terraquakeapi.com/v1/earthquakes/today?limit=50"
```

```javascript JavaScript
const response = await fetch(
  'https://api.terraquakeapi.com/v1/earthquakes/today?limit=50'
);
const data = await response.json();

if (data.success) {
  console.log(`Total earthquakes today: ${data.totalEarthquakes}`);
  data.payload.forEach(quake => {
    const { mag, place, time } = quake.properties;
    console.log(`${mag} - ${place} at ${time}`);
  });
}
```

```python Python
import requests

response = requests.get(
    'https://api.terraquakeapi.com/v1/earthquakes/today',
    params={'limit': 50}
)
data = response.json()

if data.get('success'):
    print(f"Total earthquakes today: {data.get('totalEarthquakes')}")
    for quake in data.get('payload', []):
        props = quake['properties']
        print(f"{props['mag']} - {props['place']} at {props['time']}")
```
</CodeGroup>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="code" type="integer">
  HTTP status code (200 for success)
</ResponseField>

<ResponseField name="status" type="string">
  HTTP status message ("OK" for success)
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable message: "Earthquake events for today"
</ResponseField>

<ResponseField name="payload" type="array">
  Array of GeoJSON Feature objects. Each feature contains:
  
  **Properties:**
  - `eventId` (integer) - Unique event identifier
  - `originId` (integer) - Origin identifier
  - `time` (string) - ISO 8601 timestamp
  - `author` (string) - Reporting agency
  - `magType` (string) - Magnitude type (ML, Mw, etc.)
  - `mag` (float) - Magnitude value
  - `magAuthor` (string) - Magnitude author
  - `type` (string) - Event type ("earthquake")
  - `place` (string) - Location description
  - `version` (integer) - Version number
  - `geojson_creationTime` (string) - GeoJSON creation time
  
  **Geometry:**
  - `type` (string) - Always "Point"
  - `coordinates` (array) - [longitude, latitude, depth in km]
</ResponseField>

<ResponseField name="meta" type="object">
  Request metadata containing method, path, and timestamp
</ResponseField>

<ResponseField name="totalEarthquakes" type="integer">
  Total count of earthquakes that occurred today
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination details with page, totalPages, limit, and hasMore
</ResponseField>

## Implementation Details

- **Date Range:** Current day from 00:00:00 to 23:59:59 UTC
- **Timezone:** All times are in UTC
- **Time Query:** `starttime={dateStr}T00:00:00&endtime={dateStr}T23:59:59`
- **Sorting:** Default sort is by time (most recent first: `-time`)
- **Pagination:** Manual pagination applied after fetching all today's events
- **Data Source:** INGV API with GeoJSON format
